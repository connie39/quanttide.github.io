公司简介

数据工匠成立于 2014 年，是由具兼具统计学教育背景与IT研发从业经历人员组成的创业团队，兼具大数据技术与数据科学思维能力。在为企业、政府提供产品与服务的过程中，总结积累从业务理解到落地实施的整套成熟方案，实现技术知识与领域知识的高效对接。

秉承工匠精神，细致严谨的将业务逻辑抽象为数据逻辑，设计分析流程；专注精细化挖掘，倾诉数据，聆听需求，使数据服务从肤浅走向深刻；跨越学术研究与实际生产的鸿沟，推动知识的价值预期实现；以实际问题为导向沉淀产品、精益迭代，求索效率与定制的最优解；开展技术培训、根植数据行业，注重数据科学的知识传承。

发展愿景

我们偏爱灵活可伸缩的技术架构及数据分析方式，尝试设计面向范式自由的各类型数据资源的数据分析产品，为客户解决实际数据分析问题。成为解决方案、实施咨询、应用开发、教育培训等相关服务提供商，从推动技术进步和数据分析思维落地两方面着手，让数据成为真实的高附加值资源。

工作机会

数据分析师、数据科学家、数据挖掘分析师方向

职位描述

  • 实现业务理解和对业务的数据分析建模；
  • 参与公司大数据产品建设，提供数据分析功能模块的设计方案；
  • 配合研发部门完成数据模型的搭建与实现；
  • 探索并归纳业务所涉行业的数据及指标，为优化产品的精细化设计提供支持；

职位要求

  • 统计学、计算机、数学等相关专业，本科以上学历；
  • 良好的逻辑分析能力及业务敏感性；
  • 较强的沟通能力，能发掘、梳理用户数据分析的需求与模板；
  • 具有良好的文档、报告书写能力；
  • 熟悉常见的数据挖掘和机器学习算法；
  • 了解 Python 、 R 编程， Hadoop 、 Spark 等分布式技术者优先；

Web 开发工程师、服务器端开发工程师方向

职位描述

  • Web 相关产品功能设计、开发与实现；
  • 使用 Web 技术，创建用户友好、符合标准、跨平台及跨浏览器的应用；
  • Web 层面数据可视化的实现与应用

技术路线

  • HTTP 协议基本原理；
  • PHP 开发框架 Laravel 、 Yii 等；
  • Git 的使用以及基本代码库管理；
  • LNMP 环境和 Linux 命令及使用；
  • 服务器端基本生产环境配置与管理；
  • 常用关系型数据数据库的使用与管理；
  • 熟悉 Javascript 、HTML 、 CSS ，了解常用前端框架使用；
  • 了解 Shell 、 Python 编程优先；

大数据开发工程师、数据分析系统工程师方向

职位描述

  • 参与大数据分析、处理、数据挖掘等系统的设计和开发；
  • 针对具体业务，与数据分析人员配合设计并开发算法；
  • 负责 ETL 逻辑、OLAP 设计和数据挖掘算法设计；
  • 应用各种数据挖掘技术、机器学习进行数据分析与数据挖掘；

技术路线

  • 统计学、计算机、数学等相关专业，本科以上学历；
  • 能熟悉使用 Python、 R 、Java 、Scala 、 C/C++ 或其他任一编程语言，能快速建立模型并对模型进行验证；
  • 拥有优秀的分析问题和解决问题的能力，能主动发现并解决问题；
  • 熟悉常见的数据挖掘和机器学习算法，精通数据仓库的相关知识；
  • 有大数据分析和 BI 系统研发和实施经验者优先；
  • 了解大规模分布式系统、海量数据处理、实时分析等方面的算法设计优先；
  • 有 Hadoop 、 Spark 等相关分布式系统使用经验优先；
